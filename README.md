# Shell, CDE, and TWM dotfiles

Dotfiles I use for my computers, mostly just shell, CDE and TWM configurations

> [!WARNING]
> This WILL overwrite a whole lot of your configurations
> Run with this in mind, otherwise just manually copy what you want.
> Please backup your files before installing!

## To install

> [!NOTE]
> Installation script will ONLY run on Arch-based Linux distributions.
> It will automatically install yay and CDE as needed, as well as a few other programs

```bash
git clone https://github.com/Cath0deGayTube/dotfiles
cd dotfiles
./install.sh
```
