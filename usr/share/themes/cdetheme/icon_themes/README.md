# CDE-Style Icon Themes

This directory contains icon themes that complement the CDE/Motif desktop look.

## Bundled: NsCDE Icon Theme

The **NsCDE** icon theme is bundled with this project and installed automatically
on first run to `~/.local/share/icons/NsCDE/`.

- **Source:** https://github.com/NsCDE/NsCDE
- **License:** GPL-3.0
- **Icons:** ~7000 icons in freedesktop.org standard structure
- **Sizes:** 16x16, 22x22, 24x24, 32x32, 48x48, 256x256
- **Inherits:** hicolor, breeze, mate, oxygen

## Optional: CDE-Reborn Icon Theme

**CDE-Reborn** by missTell is an excellent CDE-style icon theme designed
specifically for retro CDE/Motif desktop setups. It uses 48px icons which
pair perfectly with this panel.

To install manually:
1. Download `CDE-Reborn.tar.gz` from https://www.pling.com/p/1920644/
2. Extract to `~/.local/share/icons/CDE-Reborn/` or `~/.icons/CDE-Reborn/`
3. Select "CDE-Reborn" in XFCE Appearance Settings, or restart the panel
   (the panel will auto-detect and prefer it if installed)

## Optional: SGI-Enhanced (IRIX-Style)

For an SGI IRIX look instead of CDE, the **sgi-enhanced** theme provides
IRIX-inspired icons, cursors, and wallpapers for XFCE.

- **Source:** https://github.com/jujum4n/sgi-enhanced
- **License:** GPL
- **Note:** Icons are loose PNG/ICO files, not a freedesktop theme.
  You may need to restructure them manually.

## Switching Icon Themes

In XFCE: Settings > Appearance > Icons tab
In LXQt: Preferences > LXQt Settings > Appearance > Icon Theme
